## Skills Originais

Todas as skills do framework (`/spec`, `/plan`, `/reflect`, `/implement`, `/fix`, `/debug`, `/refactor`, `/review`, `/status`, `/learn`, `/context`, `/socrates`, `/estimate`) são **criações originais** do projeto Spec-Driven Agent Framework, salvo onde indicado.

## Ponytail (Ponytail Skill + Ponytail Review)

**Autoria:** Criação original do projeto Spec-Driven Agent Framework.

**Inspiração conceitual:**

| Conceito | Origem |
|----------|--------|
| **YAGNI** (You Ain't Gonna Need It) | Extreme Programming — Ron Jeffries, Kent Beck |
| **KISS** (Keep It Simple, Stupid) | Engenharia — Kelly Johnson (Lockheed Skunk Works) |
| **Princípio do menor esforço** | Linguística — George Kingsley Zipf |
| **Lazy evaluation** | Ciência da computação — Christopher Strachey |

O nome "Ponytail" é uma **marca original** deste framework, representando o arquétipo do "dev sênior preguiçoso" que resolve problemas com o mínimo de complexidade necessária.

## Samantha Agent

**Autoria:** Criação original do projeto Spec-Driven Agent Framework.

Inspirado em fluxos de automação e produtividade de equipes de desenvolvimento ágil. Nome "Samantha" é uma **designação original** deste framework.

## Reference Guides (10)

Todos os guias de referência são **compilações originais** de boas práticas da indústria, organizadas especificamente para consumo por agentes LLM:

- `bash-best-practices.md` — Shell scripting patterns (inspirado em Google Shell Style Guide, O'Reilly bash cookbook)
- `ci-cd-patterns.md` — GitHub Actions, Turborepo (baseado em documentação oficial)
- `documentation-templates.md` — README, ADR, CHANGELOG (baseado em Keep a Changelog, Conventional Commits)
- `git-workflows.md` — Git branching, Conventional Commits (baseado em documentação oficial Git)
- `project-structure.md` — Monorepo layouts (baseado em Turborepo, Nx conventions)
- `testing-patterns.md` — TDD, mocking (inspirado em Kent Beck TDD, Martin Fowler)
- `security-best-practices.md` — OWASP Top 10, auth patterns
- `performance-optimization.md` — Core Web Vitals, caching (baseado em Web.dev, MDN)
- `samantha-audit-checklists.md` — Criação original
- `samantha-patterns.md` — Criação original
