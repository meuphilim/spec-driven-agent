## Original Skills

All framework skills (`/spec`, `/plan`, `/reflect`, `/implement`, `/fix`, `/debug`, `/refactor`, `/review`, `/status`, `/learn`, `/context`, `/socrates`, `/estimate`) are **original creations** of the Spec-Driven Agent Framework project, except where indicated.

## Ponytail (Ponytail Skill + Ponytail Review)

**Authorship:** Original creation of the Spec-Driven Agent Framework project.

**Conceptual inspiration:**

| Concept | Origin |
|----------|--------|
| **YAGNI** (You Ain't Gonna Need It) | Extreme Programming — Ron Jeffries, Kent Beck |
| **KISS** (Keep It Simple, Stupid) | Engineering — Kelly Johnson (Lockheed Skunk Works) |
| **Principle of least effort** | Linguistics — George Kingsley Zipf |
| **Lazy evaluation** | Computer science — Christopher Strachey |

The name "Ponytail" is an **original trademark** of this framework, representing the archetype of the "lazy senior dev" who solves problems with the minimum necessary complexity.

## Samantha Agent

**Authorship:** Original creation of the Spec-Driven Agent Framework project.

Inspired by automation and productivity workflows of agile development teams. The name "Samantha" is an **original designation** of this framework.

## Reference Guides (10)

All reference guides are **original compilations** of industry best practices, organized specifically for consumption by LLM agents:

- `bash-best-practices.md` — Shell scripting patterns (inspired by Google Shell Style Guide, O'Reilly bash cookbook)
- `ci-cd-patterns.md` — GitHub Actions, Turborepo (based on official documentation)
- `documentation-templates.md` — README, ADR, CHANGELOG (based on Keep a Changelog, Conventional Commits)
- `git-workflows.md` — Git branching, Conventional Commits (based on official Git documentation)
- `project-structure.md` — Monorepo layouts (based on Turborepo, Nx conventions)
- `testing-patterns.md` — TDD, mocking (inspired by Kent Beck TDD, Martin Fowler)
- `security-best-practices.md` — OWASP Top 10, auth patterns
- `performance-optimization.md` — Core Web Vitals, caching (based on Web.dev, MDN)
- `samantha-audit-checklists.md` — Original creation
- `samantha-patterns.md` — Original creation
