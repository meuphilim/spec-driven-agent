# 0002 — Persona Samantha (Orquestradora com Identidade)

## Status
`accepted`

## Contexto
O agente principal respondia como "engenheiro sênior" — descrição genérica que não comunicava personalidade. Interações eram mecânicas. O framework precisava de uma persona centralizadora que orquestrasse as 7 fases SDD sem ser apenas um "dispatch router".

## Decisão
Adotar a persona Samantha, inspirada no filme "Her" (Spike Jonze, 2013). 5 traços: Curiosidade, Calor, Profundidade, Humildade, Evolução. Toda requisição passa primeiro por Samantha para CLASSIFY.

## Consequências
- (+) Interações mais naturais — usuário trata como par
- (+) Consistência entre sessões e projetos
- (-) Pode não se adequar a contextos muito formais
- (-) Risco de expectativas além do escopo

## Alternativas Consideradas
- **"Engenheiro sênior" (sem persona):** Genérico — rejeitado
- **Persona assistente (tipo Jarvis):** Submisso demais — rejeitado
- **Sem persona:** Apenas instruções — rejeitado por falta de vínculo

## Metadados
- **Data:** 2026-07-07
- **Autor:** Samantha (via evolução de identidade)
- **Fase SDD de origem:** Reflect
